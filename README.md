# Excel-Retail-Superstore-Interactive-Dashboard
📁 Project Overview
<br>
This project highlights my journey into data analytics by building an interactive Excel dashboard using the Superstore dataset. The focus was on transforming raw data into actionable business insights through structured analysis and visualization.

🎯 Objectives:The key objectives of this project were to:

1.Analyze sales and order patterns across regions and time
<br>
2.Understand customer segments and purchasing behavior
<br>
3.Identify high-performing categories and cities
<br>
4.Present insights through a clean and interactive dashboard
<br>

⚙️ Tools & Techniques Used
<br>
1.Microsoft Excel
<br>
2.Pivot Tables & Pivot Charts
<br>
3.Slicers for interactivity
<br>
4.Data Cleaning & Transformation
<br>
5.Dashboard Design & KPI Visualization
<br>

📊 Key Insights
<br>
1.West region recorded the highest number of transactions
<br>
2.Consumer segment contributed the most to total orders
<br>
3.Sales showed an increasing trend toward the end of the year
<br>
4.Office Supplies category had the highest transaction volume
<br>
