# Excel-Retail-Superstore-Interactive-Dashboard
Interactive Excel dashboard built using Superstore data. Includes KPI tracking (Sales, Quantity, Transactions), pivot tables, slicers, and visual insights on regions, categories, and trends to support business decision-making.
